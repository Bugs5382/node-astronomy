    Class ConstellationVisibility

    Observer-aware view of one constellation. Tells you whether the constellation's centroid is currently above the horizon and where it sits in the sky as alt/az.

    Compass-direction filtering is intentionally not part of this API — the consuming app should compute that from the alt/az we return.

    Since

    0.2.0

    Example

    const v = new ConstellationVisibility({
      constellation: "Orion",
      latitude: 40.7128,
      longitude: -74.0060,
      time: new Date("2026-01-15T03:00:00Z"),
    });
    v.isVisible();   // boolean — above horizon
    v.centerAltAz(); // { altitude, azimuth }

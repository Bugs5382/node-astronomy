    Class StarTimes

    Observer-aware view of a single named star — alt/az, rise/set, arc/track, peak.

    For circumpolar stars (e.g. Polaris from northern temperate latitudes), nextRise() and nextSet() return undefined and arc() returns []. Use dailyTrack to sample the star's 24-hour path in that case.

    Since

    0.2.0

    Example

    const polaris = new StarTimes({
      star: "Polaris",
      latitude: 40.7128,
      longitude: -74.0060,
      timezone: "America/New_York",
      time: new Date("2026-04-30T22:00:00Z"),
    });
    polaris.isCircumpolar(); // true at NYC
    polaris.position();      // { altitude ≈ 40°, azimuth ≈ 0° }

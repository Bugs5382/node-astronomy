    Class PlanetTimes

    Observer-aware view of a single planet — alt/az, rise/set, arc/track.

    Use this with any planet except Earth (which has no meaningful rise/set from itself).

    Since

    0.2.0

    Example

    import { PlanetTimes } from "node-astronomy";

    const mars = new PlanetTimes({
      planet: "mars",
      latitude: 40.7128,
      longitude: -74.0060,
      timezone: "America/New_York",
      time: new Date("2026-04-30T22:00:00Z"),
    });

    mars.position();
    mars.nextRise();
    mars.arc({ samples: 60 });

    Class Constellation

    Snapshot of one of the 88 IAU constellations.

    Constellation data is static — the boundaries don't move on human timescales. The class exposes the IAU name, abbreviation, meaning, boundary GeoJSON, and pre-computed centroid (RA / Dec).

    Since

    0.2.0

    Example

    const orion = new Constellation({ constellation: "Orion" });
    orion.abbreviation; // "Ori"
    orion.centroid;     // { ra: ~83°, dec: ~3° }

    • new Constellation(input: string): Constellation

      Build a Constellation snapshot.

      Accepts either:

      • A string with a canonical IAU Latin name (any case, e.g. "Orion", "orion", "ORION"), an IAU 3-letter abbreviation (e.g. "Ori", "UMa"), or an asterism / English alias (e.g. "big dipper", "northern cross", "plough").
      • The original { constellation, time? } object form.

      Unknown input throws with a "did you mean…?" suggestion drawn from the canonical names + alias table.
